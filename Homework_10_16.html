Definitions

Computer Terms:

Browser:
The software on a computer that allows user to browse the internet.

Web Server:
A location where files for a website are stored.

HTML:
Hyper Text Markup Language, the language needed to communicate with a computer.

CSS:
Stands for cascading style sheet. Software used to enhance a web page.

Scripting Language:
A programming language that causes an action to be performed automatically, sometimes based on a set of criteria.

Http:
Hyper Text Protocol, used to transfer data over the internet.

FTP:
File Transfer Protocol, used when transferring files on the web.

Secure FTP:
Using incription methods to create a more secure file transfer.

FTP Client:
Software needed to transfer files between a server and a computer over the internet. It must be installed on your computer to make this work.

Domain:
Names given to identify a particular web page.

URL:
Uniform Resource Locator, a domain name is part of a URL. Aids in locating specific web pages.

HTML Tag:
Set of characters giving formatting direction to a web page.

Source Code:
The code typed into a computer by a human in understandable type.

Responsive Design:
Is the approach that suggests that design and development of a web page should respond to the userâ€™s behavior and environment. This would include screen size and the platform.

Web hosting company:
Provide space on a server for a client's web pages. There are different levels of services offered. Some manage the pages and their security, others provide the space but website owners need to keep their sites secure.

Text Editor:
Software that edits plain text. Notepad & Word are types of text editors.

Graphics:
Images displayed on a computer screen. A picture is a graphic, sometimes graphics contain numbers.

When you log into your web space with an FTP client, why will you have a public_HTML folder in there? What will happen if you delete this folder?
The Public_html folder is the root location for your website. You will put all files associated with the site in this folder. This is the folder associated with your domain name. If this is deleted, your site cannot be found using the name given to it.
